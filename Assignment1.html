Web Development 2022

Static Website: Hunter College Undergraduate Catalog

The Hunter College Undergraduate Catalog is a static website. This is because the website is written with HTML, each webpage is a separate document, and there are no databases or external files drawn upon. I like that this website is simple and easy to navigate.


Dynamic Website: Wikipedia

Wikipedia is a dynamic website. This is because the website is written with more complicated code (PHP), it has more functionality, and databases are used. I like that this website has an abundance of information that is contributed by many people.
